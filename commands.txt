1. Django-admin startproject projectname -> to start a projectname
2 .python3 manage.py startapp appname -> to create a module
3. add app to settings.py of projectname module under INSTALLED_APPS
4. python3 manage.py runserver -> to run Django project


# to create a model
1. create class in model in app
2. python3 manage.py makemigrations -> to make migrations, populates migrations folder of all the apps
3. python3 manage.py migrate -> run all the migrations that has n;t been executed yet
